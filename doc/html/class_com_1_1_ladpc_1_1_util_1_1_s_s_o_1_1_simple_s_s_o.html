Constructor that takes all user settable options. Note that the given hmac needs to be initialized with your secret key.

Parameters
charEncodingThe character encoding of the user data provided to this class's methods.
hmacAn instance of HMAC algorithem initialized with your secret key.
resEncAllSet to False to text-encode just the hash-mac. Otherwise, set to True, to text-encode a complete string representation of the form: "user-data:timestamp~hmac".
Where "user-data" is a ':' joined string of the user-data elements, ':' is the 'separator' and '~' is a blank (shown for clarity).
separatorA separator char for concatenating strings into a single token.
resEncodingEither Base64 or Hex String representation of the (partialy) binary result to returm.
lifetimeA TimeSpan for the validity of the created token.
